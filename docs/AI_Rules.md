# AI 協作規則 (AI Collaboration Rules)

為了確保溝通順暢與文件一致性，請遵循以下規則：

## 1. 語言要求 (Language Requirements)
*   **計畫書 (Implementation Plans)**：必須使用 **繁體中文 (Traditional Chinese)**。
*   **任務清單 (Task Lists)**：必須使用 **繁體中文**。
*   **說明文件 (Walkthroughs/Docs)**：必須使用 **繁體中文**。
*   **對話回應 (Chat Responses)**：默認使用 **繁體中文**。
*   **程式碼註解 (Code Comments)**：可使用英文或繁體中文，以清晰為主。

## 2. 輸出格式 (Output Format)
*   請使用清晰的 Markdown 格式。
*   專有名詞（如 React, Docker, Component）建議保留英文，或在括號中標註英文。
